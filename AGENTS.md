# AGENTS.md

This repository is a Godot-based visualization/control-room project for a robotics system.

Priorities:
- Understand current implemented behavior before proposing changes.
- Prefer facts from code over assumptions from old comments or stale docs.
- Keep documentation concise and technical.
- Do not rewrite runtime code unless explicitly asked.
- When documenting MQTT, separate current implemented behavior from recommended cleanup.
- Assume the visualization is part of the same system as Minone and PolyMap.
- Note simulation/test artifacts if present, but do not overemphasize prototype history.
